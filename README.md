# Terraform Mastery Journey

This repository documents my hands-on Terraform and DevOps learning journey where I built real AWS infrastructure projects step-by-step while understanding how Infrastructure as Code works in practical environments.

The focus of this journey is not only writing Terraform code, but also learning:
- Infrastructure automation
- Cloud architecture
- State management
- Modular design

- Security integrations
- Monitoring and drift detection

Each day contains a dedicated project with its own detailed README explaining:
- Concepts learned
- Project architecture
- File structure
- Commands used
- Problems faced
- Troubleshooting steps
- Real-world DevOps practices
