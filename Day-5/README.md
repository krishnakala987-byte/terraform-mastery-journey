# Terraform Provisioners to Production Deployment (Day 5)

This project demonstrates how to provision AWS infrastructure using Terraform and deploy a Flask application on EC2 using industry best practices.

## Topics Covered
- Terraform modules (VPC, EC2)
- user_data vs provisioners
- EC2 + Flask deployment
- Debugging using logs
- Python virtual environments (PEP 668 fix)
- Nginx reverse proxy
