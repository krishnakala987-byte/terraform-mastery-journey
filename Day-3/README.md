# Terraform Modules (Day 3)

This project focuses on converting a basic Terraform setup into reusable modules like real-world DevOps projects.

## Topics Covered

- What are Terraform modules
- Why modules are used in industry
- Creating reusable infrastructure components
- Splitting code into EC2, S3, and Security Group modules
- Connecting modules using inputs and outputs
- Remote backend with S3 and state locking
- Troubleshooting backend and locking issues
